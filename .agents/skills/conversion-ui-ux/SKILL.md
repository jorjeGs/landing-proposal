# Goal
Enhance user experience flow to reduce friction, build trust, and increase conversion rates.

# Instructions
1. Simplify forms and interactive flows to the minimum necessary actions.
2. Keep primary call-to-actions prominent and visually distinct from secondary elements.
3. Place trust-building indicators (testimonials, rating badges, security icons) near critical buttons (e.g., checkout/sign-up).
4. Provide real-time validation and feedback on user inputs to prevent errors before submission.
5. Reduce visual distraction on checkout, signup, or registration layouts.

# Input
Existing user sign-up/checkout flow, user feedback, or conversion metrics.

# Output
Refined form layouts, visual CTA enhancements, and streamlined flow architectures.

# Best Practices
- Use clean microcopy to explain why sensitive information is collected.
- Focus on immediate load times and avoid layout shifting during interaction.
- Track conversions using analytics tags or tracking events.
