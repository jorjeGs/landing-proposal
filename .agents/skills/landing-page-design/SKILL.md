# Goal
Design conversion-focused, visually engaging, and narrative-driven landing pages.

# Instructions
1. Structure the layout with a clear hierarchy: Hero with clear headline, Social Proof, Features, Benefits, Testimonials, FAQs, and a final Call-to-Action (CTA).
2. Use compelling, large-scale typography paired with a clean body font.
3. Create strong contrast for primary CTAs and place them strategically at multiple scroll depths.
4. Integrate visual assets (illustrations, product shots, generated image assets) to tell a cohesive story.
5. Keep page load performance optimal by using compressed images and clean structural layout.

# Input
Product description, target audience, brand aesthetic, and copywriting briefs.

# Output
Responsive landing page structure, styling tokens, and layout implementation.

# Best Practices
- Keep copy readable and concise. Avoid dense walls of text.
- A/B test layout structures, CTA placement, and button copy to maximize conversion.
- Ensure the landing page is responsive and performs exceptionally on mobile.
